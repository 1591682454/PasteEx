# PasteEx
[![](https://img.shields.io/badge/platform-windows-lightgrey.svg)](https://www.microsoft.com/download/details.aspx?id=30653)
[![Github All Releases](https://img.shields.io/github/downloads/huiyadanli/PasteEx/total.svg)](https://github.com/huiyadanli/PasteEx/releases)

把剪贴板的内容粘贴为文件。

可以自动识别图片的扩展名，以及自定义文本扩展名规则。

支持透明 PNG 与 GIF 动态图的粘贴。

[:zap:最新版本 v1.0.3.5 下载](https://github.com/huiyadanli/PasteEx/releases/download/1.0.3.5/PasteEx.v1.0.3.5.zip)

## 使用方法
需要 [.NET Framework 4.5](https://www.microsoft.com/zh-cn/download/details.aspx?id=30653) 的支持。

先把本程序置于一个空文件夹内（运行过程中会在当前目录下释放配置文件）。

然后直接运行，根据提示添加右键菜单即可（添加菜单时会有 UAC 提示，放行即可）。

卸载方式：纯绿色便携软件，只要在设置页面移除右键菜单，然后删除软件本体即可。
